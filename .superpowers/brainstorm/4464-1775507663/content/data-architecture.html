Key Design Decisions

Version Pinning

Characters reference content at a specific version via character_content_refs. DM updates create a new version row; players opt in to adopt it.

Content Scoping

Every content_definition has a scope: platform (SRD), user (personal), or campaign (shared). RLS enforces visibility.

Effects as Data

Content carries an effects array — each effect is a typed operation like {stat: "movement_speed", op: "add", value: 10} or a formula expression.

Characters are Lean

Characters store base values + choices. Derived stats (AC, modifiers, save DCs) are computed at runtime by the expression engine from base_stats + content effects.
