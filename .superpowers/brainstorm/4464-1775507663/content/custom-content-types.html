How DMs create novel mechanics like Ship Roles and Criminal Org Ranks

Custom Content Type Definition

{
  "name": "Ship Roles",
  "allow_multiple": false,
  "entry_conditions": [

  "has_progression": true
}

Trigger Types

Stat-Gated (auto)

Auto-unlocks when any stat reaches a threshold. Uses the same condition syntax as content prerequisites. Can chain multiple conditions with AND logic.

Examples: level >= 8, gold >= 500, max_hp >= 50, wisdom >= 18, custom_reputation >= 10

Manual

Player or DM explicitly advances to the next tier. Surfaces as a button on the character sheet: "Advance to [next tier name]". DM can lock manual advancement to DM-only if desired.

Examples: org promotions, story milestones, quest completions

Each tier in a progression track independently chooses its trigger type — so you can mix auto and manual within the same track.
