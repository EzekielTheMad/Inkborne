Row Level Security Policies
content_definitions visibility:
• platform → everyone
• personal → owner only
• shared → owner + members of campaigns in content_shares
characters visibility:
• private → owner + campaign DM
• campaign → owner + all campaign members
• public → everyone
campaigns:
• Visible to members only
• Editable by DM (owner) only
custom_content_types:
• personal → owner only
• shared → owner + campaigns in content_type_shares
Edit Access
• Characters → owner only (DM can view but not edit)
• Content definitions → owner only (sharing grants read, not write)
• Custom content types → owner only
• Campaigns → DM (owner) manages settings; members can leave
• Content shares → any campaign member can share their own content
